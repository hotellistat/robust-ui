# Progress

| Prop    | Description                        | Accepted Values | Default |
| :------ | :--------------------------------- | :-------------- | :------ |
| `value` | Current value                      | -               | 0       |
| `max`   | Maximum value for the progress bar | -               | 100     |

Progress

```vue
<Progress value="50" max="100" />
```
